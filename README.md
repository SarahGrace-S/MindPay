# 💡 MindPay

MindPay is a smart payment decision web application that helps users think before they spend money.

## 🚀 Features

- Classify payments as:
  - Essential
  - Non-Essential
  - Emergency
- Provides instant decision feedback
- Simple and clean user interface
- Backend API using Node.js & Express

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB (optional)

## ⚙️ How it works

1. User enters payment details (amount & purpose)
2. Selects a category
3. System analyzes and gives a response:
   - ✅ Approved
   - ⚠️ Warning
   - 🚨 Emergency alert
